FakeOnlinePlayer

class FakeOnlinePlayer(offlinePlayer: OfflinePlayer, location: Location) : Player, OfflinePlayer

A Player which will throw an UnsupportedOperationException when methods which aren't supported by OfflinePlayer are called.

This Player is also granted access to the hasPermission method via Vault.
