sendDestructionPacket

fun <Error class: unknown class>.sendDestructionPacket(entityId: Int, stage: Int)

Sends the ClientboundBlockDestructionPacket to all players in a 1-chunk-range with the given entityId and breaking stage. Might not work with some Nova blocks.

See also

Block.setBreakStage

Parameters

entityId

The id of the entity breaking the block

stage

The breaking stage between 0-9 (both inclusive). A different number will cause the breaking texture to disappear.
